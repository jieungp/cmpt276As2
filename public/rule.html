Rule:
- Add: name -> max. 10 characters, size and height -> numbers, type -> max. 3 characters.
- Update: Updating Name and Type -> row wanting to change, Attribute type -> name of column, New value -> value you want to change to
- Delete: Must enter both Name and type
- Database: Shows the whole database collected
- Individual: Allows you to choose single database to view

- In the DATABASE, PRIMARY KEY is Union of 'NAME' and 'TYPE'
- Due to PRIMARY KEY, You CANNOT enter values that BOTH 'NAME' and 'TYPE' already existing in the DATABASE
- Entering wrong values in 'DELETE' and 'UPDATE' DO NOT make any changes
